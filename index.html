Batatabit, es una app para intercambio de criptomedas, fué creada con html y css con un diseño en Figma y comenzando desde mobile first. Adaptando luego los diseños responsives a pantallas más grandes, usando scroll en pantallas para celulares, botones flotantes, así como variables para los diferentes estilos.

Burger Queen es una interfaz diseñada para locales de comida, que permite facilitar el flujo que se lleva a cabo desde la toma de pedidos que realiza el mesero, pasando por la cocina y hasta la entrega de estos al cliente.
